Manual Equipe - PWA Instalável com Splash Premium

Arquivos principais:
- index.html: app com splash premium animado aplicado
- manifest.webmanifest: manifesto PWA
- sw.js: service worker
- icons/: ícones do PWA baseados no 5º ícone escolhido
- splash-preview-premium.png: imagem de referência do splash aprovado
- backup-original.html: backup do arquivo original enviado
- backup-icone-5-sem-splash.html: backup da versão anterior sem splash

Para publicar no GitHub Pages, suba todo o conteúdo desta pasta para o repositório.
Se o celular continuar mostrando versão antiga, desinstale o app/atalho anterior e limpe cache do navegador antes de reinstalar.
