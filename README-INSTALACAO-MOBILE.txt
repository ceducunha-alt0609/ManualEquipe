CORREÇÃO PWA MOBILE - Manual Equipe

Arquivos importantes:
- index.html
- manifest.json
- manifest.webmanifest
- sw.js
- pasta icons/ completa

Para GitHub Pages:
1. Suba TODOS os arquivos na mesma pasta.
2. Acesse por HTTPS, não por arquivo local.
3. No celular, abra a URL final do GitHub Pages.
4. Se aparecer só atalho, remova o app/atalho antigo, limpe dados/cache do site e abra novamente.
5. Aguarde a página carregar uma vez inteira, feche a aba, abra de novo e tente Instalar app.
